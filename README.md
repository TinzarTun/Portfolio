# 🌟 Tinzar Tun Portfolio

This is the **personal portfolio website** of **Tinzar Tun**, showcasing **projects, skills, and contact information**. The site is fully responsive and supports **dark/light mode**, built with **HTML, CSS, MDBootstrap, and Font Awesome**.

---

## 📌 Overview

The portfolio website highlights:

- **About Me:** Personal introduction and professional goals.
- **Skills:** Frontend, backend, database, tools, and soft skills.
- **Projects:** Top and all projects with GitHub links.
- **Contact:** Direct email and social media connections.
- **Dark/Light Mode:** Toggleable for user preference.

The site demonstrates clean design, responsiveness, and interactive UI elements like fade-in animations and hover effects.

---

## 🚀 Features

### General

- Responsive layout (desktop, tablet, mobile)
- Dark/Light mode toggle
- Smooth scroll animations
- Accessible navigation bar
- Footer with social links

### Projects Section

- Categorized project cards
- Hover effects with details and GitHub links
- “View All Projects” page with filters (All, Recent, In Progress)
- Highlighted key projects like **Bus Ticket System**, **Lost & Found**, **E-Commerce System**, **Lavender Calculator**, **JSON Server API Test**, etc.

### Contact Section

- Email link for direct communication
- Social media links: LinkedIn, GitHub, Telegram, Viber

---

## 🛠 Tech Stack

**Frontend:** HTML5, CSS3, MDBootstrap, Font Awesome  
**UI/UX:** Responsive Design, Dark/Light Mode, Animations  
**Deployment:** Static HTML, easily deployable on GitHub Pages or any web server

---

## 💻 Installation / Running Locally

1. Clone the repository:

```bash
git clone https://github.com/TinzarTun/Portfolio.git
```

2. Open `portfolio.html` in your browser.

> No backend required; the website runs entirely on **static HTML/CSS/JS**.
